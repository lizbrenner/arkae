# Arkae Design System

Welcome to the Arkae Design System documentation! This Storybook showcases all components, tokens, and patterns that make up the Arkae design language.

## What is Arkae?

Arkae is an AI-native design system built on [Base UI](https://base-ui.com/) primitives. It provides a comprehensive set of accessible, customizable components and design tokens for building modern web applications.

## Getting Started

### Installation

Install the complete design system:

```bash
npm install @arkae/library
```

Or install individual packages:

```bash
npm install @arkae/primitives @arkae/tokens
```

### Usage

```tsx
import { Button } from '@arkae/library';
import '@arkae/library/styles.css';

function App() {
  return (
    <Button variant="primary" size="md">
      Click me
    </Button>
  );
}
```

## Package Structure

- **@arkae/tokens** - Design tokens (colors, typography, spacing, etc.)
- **@arkae/primitives** - Base UI components (Button, Input, Dialog, etc.)
- **@arkae/compositions** - Complex composed components
- **@arkae/library** - Complete design system (umbrella package)
