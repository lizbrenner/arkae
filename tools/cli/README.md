# @arkae/cli

Command-line interface for Arkae design system.

## Installation

```bash
npm install -g @arkae/cli
```

## Usage

```bash
# Generate a new theme
arkae theme generate

# Create a component
arkae component create

# Analyze accessibility
arkae a11y analyze

# Evaluate usage
arkae evaluate
```

## Commands

- `theme generate` - Generate design themes
- `component create` - Create new components
- `a11y analyze` - Analyze accessibility
- `evaluate` - Run LLM evaluations
- `init` - Initialize Arkae in a project

## Development

```bash
pnpm build    # Build the package
pnpm dev      # Watch mode
```
